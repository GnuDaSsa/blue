# projects/ — 프로젝트별 작업 기록

영상·사진·음악 프로젝트를 1프로젝트 1페이지로 관리한다.

## 작성 원칙
- 파일명은 `YYYY-MM-프로젝트명.md` 또는 `프로젝트명.md` 형식의 kebab-case.
- 프로젝트 페이지에는 목표, 레퍼런스, 사용 툴, 생성 프롬프트, 결과물, 회고를 남긴다.
- 영상 프로젝트는 반드시 [[video-workflow]]를 따른다.
- 최종 프롬프트는 `prompts/`에도 별도 파일로 저장하고 프로젝트 페이지에서 링크한다.
