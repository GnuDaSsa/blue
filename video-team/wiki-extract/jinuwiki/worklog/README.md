# worklog/ — 상세 작업 이력

`log.md`는 짧은 append-only 인덱스이고, `worklog/`는 실제 작업 과정을 자세히 기록한다.

## 작성 원칙
- 월 단위 파일을 기본으로 사용한다: `YYYY-MM.md`.
- 의미 있는 설정 변경, 프롬프트 제작, 레퍼런스 정리, 결과 분석, 실패 원인을 기록한다.
- 새 영상 업무를 시작할 때 최근 월간 로그를 먼저 훑어 이전 결정과 반복 실수를 확인한다.
- 작업이 끝나면 이번에 만든 프롬프트/결과/다음 액션을 추가한다.
