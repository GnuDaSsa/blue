# prompts/ — 최종 프롬프트 아카이브

Seedance, Suno 등 생성 도구에 실제로 넣을 최종 프롬프트와 변형안을 저장한다.

## 작성 원칙
- 파일명은 `YYYY-MM-DD-작업명-tool.md` 형식 권장.
- 원본 요청, 참고한 위키 페이지, 최종 프롬프트, 네거티브/주의사항, 결과 메모를 함께 남긴다.
- 영상 프롬프트는 [[video-workflow]]와 [[seedance-2.0]]을 참고한다.
- 음악/Suno 프롬프트는 설치된 `music-composition` 스킬을 함께 참고한다.
